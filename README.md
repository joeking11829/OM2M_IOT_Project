# NTOU Homework

### Build Docker Image of OM2M
```
cd docker/om2m_one
./docker_build.sh
```

### Run Docker Container of M2M
```
cd docker/om2m_one
./docker_run.sh
```

### Start OM2M and Node-RED ( Inside Container )
```
# execute start script
./src/start_om2m.sh
```

### OM2M & Node-RED WebUI

- mn-cse: http://127.0.0.1:8282/webpage/
